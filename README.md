# mdbook-typst-pdf

将 [mdBook](https://github.com/rust-lang/mdBook) 转换为 PDF。

目前主要用于将 [Rust 程序设计语言 简体中文版](https://kaisery.github.io/trpl-zh-cn) 转换为 PDF。
